Endpoints

Catálogo rápido

Cada bloco abaixo já vem com um exemplo copiável para uso imediato.

POST

/v1/chat/completions

Endpoint principal, compatível com OpenAI.

curl http://localhost:3000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "mimo-v2.5-pro",
    "messages": [
      {"role": "user", "content": "Responda apenas: ok"}
    ],
    "stream": false
  }'
POST

/api/chat

Compatível com clientes Ollama. Se stream não for enviado, o padrão é false.

curl http://localhost:3000/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "model": "mimo-v2.5-pro",
    "messages": [
      {"role": "user", "content": "Olá"}
    ]
  }'
POST

/api/generate

Fluxo de geração simples no formato Ollama. Se stream não for enviado, o padrão é false.

curl http://localhost:3000/api/generate \
  -H "Content-Type: application/json" \
  -d '{
    "model": "mimo-v2.5-pro",
    "prompt": "Explique este projeto em uma frase.",
    "stream": false
  }'
GET

/v1/models, /api/tags, /health

Endpoints de leitura e diagnóstico.

curl http://localhost:3000/v1/models
curl http://localhost:3000/api/tags
curl http://localhost:3000/health
